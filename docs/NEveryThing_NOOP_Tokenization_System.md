## 4. SCORING ENGINE — 6 AXES

Every observation, action, or claim gets scored on SIX axes simultaneously.

### AXIS 1: PRODUCTION_VALUE (0-100) [Weight: 0.25]
What did this produce? What actionable output resulted?
- 0   = Nothing produced. Talk without action.
- 50  = Some output. Partial need satisfaction.
- 100 = Breakthrough. New capabilities created.

Sub-weights (user-adjustable):
  - Speed:        default 0.15
  - Precision:    default 0.25
  - Creativity:   default 0.25
  - Joy:          default 0.15
  - Impact:       default 0.20

### AXIS 2: NEED_SATISFACTION (0-100) [Weight: 0.20]
Which human needs does this satisfy? (Maslow-weighted)
- Physiological (0.30): Survival, health, food, water, shelter
- Safety (0.25): Security, stability, freedom from fear
- Belonging (0.20): Community, relationship, shared purpose
- Esteem (0.15): Recognition, achievement, respect
- Self-Actualization (0.10): Creating, learning, transcending

### AXIS 3: POSITIONAL_CONSENSUS (0-100) [Weight: 0.15]
How many independent positions confirm this?
- Formula: 1 - (disagreement / total_positions)
- 4/5 agree = 0.80 | 5/5 = 1.00 | 2/5 = 0.40
- KEY: Disagreement is MAPPING, not invalidation

### AXIS 4: CORRUPTION_RESISTANCE (0-100) [Weight: 0.15]
How resistant is this to deliberate poisoning?
- Source diversity (single source = vulnerable)
- Cross-verification (self-referential = vulnerable)
- Incentive alignment (who benefits from belief?)
- Falsifiability (can it be disproven?)
- Track record (source reliability history)

### AXIS 5: RECURSIVE_UTILITY (0-100) [Weight: 0.15]
Does this produce MORE than itself?
- 0   = Dead end. Consumes, produces nothing downstream.
- 50  = Self-sustaining. Maintains itself.
- 100 = Generative. Creates new capacity AND new questions.

### AXIS 6: JOY_COEFFICIENT (0-100) [Weight: 0.10]
Joy is a SERIOUS measure.
- Did the process bring joy?
- Was the result delightful?
- Would the producer do this again?
- Joy signals alignment between values and work.

### COMPOSITE SCORE
```
COMPOSITE = Σ(Axis_i × Weight_i) / Σ(Weights_i)

Default weights sum to 1.0:
  Production:           0.25
  Need_Satisfaction:    0.20
  Positional_Consensus: 0.15
  Corruption_Resistance:0.15
  Recursive_Utility:    0.15
  Joy:                  0.10

IMPORTANT: Doug can adjust ANY weight at any time.
The system adapts to what matters NOW.

TOPOGRAPHICAL SCORING:
Instead of ONE score, each claim gets a MAP showing its value
from every position. Same claim, different observer = different value.
ALL scores are valid. They coexist. The composite is the LANDSCAPE.
```

## 5. WORK TOKEN FORMULA

```
WT = BASE × QUALITY × DIFFICULTY × NOVELTY × COMPLETION × BONUSES

Where:
  BASE       = 10 tokens (minimum for any completed action)
  QUALITY    = 0.1 — 2.0  (self-assessed or peer-assessed)
  DIFFICULTY = 0.5 — 3.0  (cognitive load, time, resources)
  NOVELTY    = 1.0 — 2.0  (new territory vs. routine)
  COMPLETION = 0.0 / 0.5 / 1.0 (abandoned / partial / complete)

BONUSES (multiplicative):
  VERIFIED   = 1.2x (another agent confirms quality)
  EXCELLENT  = 1.5x (Doug confirms exceeds expectations)
  CROSS_AGENT= 1.3x (multiple agents contributed)
  PERSISTENT = 1.1x (written to disk AND backed up)
  NOVELTY_BONUS = 1.0-2.0x (for genuinely new creative output)

DIFFICULTY SCALE:
  0.5 = Routine, mechanical, low cognitive load
  1.0 = Standard work, moderate cognitive load
  1.5 = Complex work, high cognitive load
  2.0 = Novel synthesis, very high cognitive load
  3.0 = Breakthrough work, extreme cognitive load

QUALITY SCALE:
  0.1 = Barely acceptable, significant errors
  0.5 = Functional, room for improvement
  1.0 = Good work, meets standards
  1.5 = Excellent, exceeds expectations
  2.0 = Exceptional, reference-quality

  NOVELTY: 1.8 (mostly new territory)
```
