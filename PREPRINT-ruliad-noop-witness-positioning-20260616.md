## 2. The Convergence: Four Frameworks, One Insight

### 2.1 Leibniz — Monadology (1714)

Leibniz's monads [2] are the fundamental units of reality:
- **Windowless**: no external input. Nothing enters, nothing leaves.
- **Simple**: no parts. Indestructible. Cannot be created or destroyed naturally.
- **Complete internal state**: each monad contains its entire state internally.
- **Mirroring**: each monad reflects the entire universe from its unique perspective.
- **Pre-established harmony**: all monads are synchronized without interacting. God set initial conditions so every monad's internal evolution matches every other's.
- **Appetition**: the internal principle driving the monad from one state to the next.

### 2.2 Markov — Stochastic Processes (1906)

The Markov property [3]:
- **Memoryless**: next state depends only on present state.
- **No path dependence**: how you arrived does not matter. Where you are determines where you go.
- **State contains all information**: the present encodes everything needed for the future.
- **Transition probabilities**: from any state, the distribution of possible next states is fully determined.

### 2.3 Hoffman — Conscious Realism (2010s)

Hoffman's conscious realism [4]:
- **Conscious agents are the reality**: spacetime is a user interface, a simplified dashboard.
- **Perception = sampling**: each agent perceives a simplified representation of the underlying structure.
- **No access to objective reality**: agents interact through their interfaces, not directly with the thing-in-itself.
- **The physical world is a representation**: like a desktop GUI hiding the complexity of the machine.

### 2.4 Wolfram — The Ruliad (2020)

Wolfram's ruliad [1]:
- **All possible computations entangled**: every rule, every initial condition, infinite steps.
- **Observer sampling**: each observer occupies a position in ruliad space and samples a slice.
- **Sampling IS the experience**: what you perceive is determined by where you stand.
- **Computational equivalence**: all observers are equivalent; only position differs.
